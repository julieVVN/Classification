# Classification
Objectif : <br><br>
Prédire le parti victorieux de chaque Etat US, aux élections de 2020, à partir de données socio-démographiques. <br>
Il s’agit donc d’un projet de classification binaire :<br>
  - classe 1 : Républicains <br>
  - classe 0 : Démocrates

Etapes : <br>
- Prétraiter des données pour obtenir un jeu de données exploitable <br>
- Evaluer les performances de plusieurs modèles d'apprentissage supervisé (construits via pipeline) <br>
- Optimiser les hyperparamètres des algorithmes sélectionnés afin de les améliorer <br>
- Analyser l’importance des variables
